# Rule: Research thị trường SMS Marketing

## Nguyên tắc
- KHÔNG bịa data. Crawl thực bằng WebSearch + WebFetch.
- Bắn 6-10 query parallel trong 1 message.
- Mỗi data point GIÁ phải có link nguồn + ngày thu thập.
- Không tìm được -> ghi "không thu thập được công khai".

## Data points cần thu thập mỗi provider
1. Dịch vụ: SMS brandname, OTP, bulk marketing, API
2. Giá: VND/tin theo từng loại, gói tối thiểu, chiết khấu sỉ
3. Brandname: phí đăng ký, thời gian, yêu cầu pháp lý
4. API: REST/SDK, docs quality, sandbox có không
5. Reseller/Affiliate: có chương trình không, % hoa hồng, điều kiện
6. Hợp đồng: nạp tối thiểu, prepaid/postpaid, thời hạn
7. Hỗ trợ: giờ làm việc, kênh (hotline/chat/email)
8. Website style: màu chủ đạo, tone, CTA chính, USP
