# Rule: Website & Content

## Đối tượng khách hàng
- Chủ SME, marketing manager, IT manager
- Hiểu biết kỹ thuật trung bình - không cần giải thích quá sâu API
- Quan tâm: giá rõ ràng, dễ dùng, support tiếng Việt, không bị lừa phí ẩn

## Tone viết
- Chuyên nghiệp + gần gũi + số liệu cụ thể
- Câu ngắn (<25 từ trung bình)
- Tránh: "tốt nhất", "số 1", "cam kết 100%" (vi phạm Luật Quảng cáo VN)
- Dùng: số liệu có nguồn, dẫn chứng cụ thể, demo/thử miễn phí

## CTA hiệu quả ngành SMS VN
- "Dùng thử miễn phí X tin nhắn"
- "Nhận báo giá trong 30 phút"
- "Tích hợp API trong 15 phút - có hướng dẫn tiếng Việt"
- "Tư vấn miễn phí - gọi ngay [số]"
